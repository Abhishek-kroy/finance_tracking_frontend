# Finance Tracker 💸  

Finance Tracker is a **web-based application** designed to help individuals manage their finances effectively. It allows users to create budgets, track expenses, visualize spending patterns, and generate **AI-powered financial reports** using **Gemini AI**.  

## 🚀 Features  
- **Budget Creation** – Set up and manage your monthly or yearly budgets easily.  
- **Expense Tracking** – Record, categorize, and monitor your expenses in real-time.  
- **AI-Powered Reports** – Get financial insights with AI-generated reports using **Gemini AI**.  
- **Data Visualization** – Interactive dashboards to help analyze your spending habits.  

## 🔗 Live Demo  
Check out the live application here: 👉 [Finance Tracker](https://financetrackerforproject.netlify.app/) 

## ⚙️ Requirements  
- **Node.js** – Ensure Node.js is installed on your system.  
- **MongoDB** – Used for managing the database.  
- **Firebase Authentication** – Secure login functionality.  

## 🖥️ Usage  
Once installed, users can:  
- **Create and manage budgets.**  
- **Track expenses in real-time by category.**  
- **View AI-powered financial reports.**  
- **Analyze spending trends with data visualization.**

## 🛠️ Technologies Used  
- **Frontend:** React + Tailwind CSS  
- **Backend:** Node.js + Express.js  
- **Database:** MongoDB  
- **AI Integration:** Gemini AI
